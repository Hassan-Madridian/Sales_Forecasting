# Sales Forecasting for Retail Business

## Business Problem
Retail businesses need accurate sales forecasts to optimize inventory and staffing.

## Dataset
Public retail sales dataset containing weekly sales and economic indicators.

## Approach
- Data cleaning and preprocessing
- Exploratory data analysis
- Feature engineering
- Model training and evaluation

## Models Used
- Linear Regression
- Random Forest Regressor

## Results
Random Forest outperformed Linear Regression with lower RMSE.

## Business Insights
Sales show strong seasonality and increase during holiday periods.

## Tools
Python, Pandas, Scikit-learn, Matplotlib
